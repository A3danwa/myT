# マイティールールブック

## 概要
**某学生寮で生み出された変則麻雀のルール**

「白（何も書いてない牌）」が、「何も書いてないんだったら何にでもなれるだろ」という雑な発想で、トランプでいうジョーカーのように何の牌としても使えるオールマイティーの牌として使えるというルールだったらしい。
## ゲームの準備
1, 三人で卓に座る。

2, 通常の三麻の用意をする。赤2枚。

3, 加えて一人チップを30枚分点棒入れに入れる。(A3では慣例的に白いチップ=5枚、赤・青・黄のチップ=1枚とみなしてやり取りをしている。)

4, 焼き鳥札を一人一枚用意し、卓の右手側におく。
## 半セット
公平性を期すために、起家を1回ずつ変えて3半荘を半セットとして、半セット単位で打つ（これを半セットと呼ぶ）。
## 半荘の開始
1, 点棒を一人25,000点になるように戻す。(この時チップは戻さない)

2, 焼き鳥札を表にする。
## 局の開始
1, 一人18牌ずつ山に積む

2, 通常の麻雀と同様に牌を引く

3, 山を分けたところから5番目の牌をドラ表示牌として表に捲る。
## 局の進行
通常の三麻に準拠するが以下のところでルールが変わる。

- 喰いタンアリ
- 後付けアリ
- 聴牌連荘　
- 一発裏アリ

### マイティー牌
- 白が手牌にあるとき、白をマイティー牌（いかなる牌としても見做せる牌）として扱う
- 白をマイティー牌として扱った副露はできない（白を白として使う副露は可能）
- 白を牌Aとして見做した際にフリテンとなる場合でも、別の牌Bとして見做したあがり形でフリテンでなければ牌Bとして見做したうえでロンあがりができる。
- 白をマイティー牌として扱わずに他家の切った白をロンした場合、点数が二倍になる
- 白の取り方が複数ある場合、裏、アリスをめくる前に確定させる。忘れた場合は、手牌のみで見た時の高めがあればそのように取り、変わらなければ、発覚時で安目どり。
- 手牌内（抜きドラを含む）で、同じ牌が5牌以上存在するような取り方はできない。
- 白をマイティー牌として牌Aとして扱う場合は、ドラ、アリスのカウントも牌Aとして扱う。
  マイティー牌を用いた聴牌であっても、立直後、暗槓によって待ちが変わるようなカンは出来ない。もし行った場合、発覚し次第チョンボになる。
### 白フラッシュ
白が切られた場合、切った本人以外は切った白を含めて二巡ツモ切りをしなければならない。なお、この二巡のうちは例え和了出来たとしてもツモもロンも行えなず、フリテンとなる。他家の白フラッシュの効果で白を切った場合は、白フラッシュの効果は重複し、その時点から先に白フラッシュを行った者も二巡ツモ切りとなる。白を白としてロンされた場合前述の通り支払いは2倍。別称として白焚きともいう。  
白フラッシュを行った者（以下Aと呼称する）は他家に対し、ロンや副露などを行える。白フラッシュの後、Aは牌を一巡自由に切ることができ、Aの2回目の捨牌で解除される。  
```
例)Aが親番、B,Cがそれぞれ南家、西家とする。白フラッシュをAが行うとする。

case1)白フラッシュを焚いた巡目で副露が発生

副露後、Aは牌を切り、その巡目は白フラッシュが有効でB,Cはツモ切り。その次のAの捨牌から白フラッシュは解除となる。

  

case2)白フラッシュを焚いた次の巡目で副露が発生

Aはもう1牌切っている状況であり、副露するとその捨牌から白フラッシュは解除になる。
```
### 北
- 北は、「北(きた)」と宣言し、抜きドラとして抜くことが出来る。ただし、手牌にあった場合でもドラとしてみなす。また、北は常に風牌として扱われる。
- 他家が抜いた北はポンすることが出来る。  
- 他家が抜いた北はロンすることが出来る。
- 北が抜き牌、捨て牌、ドラ表示牌などで場に4枚見えた場合、4枚目が見えた直後に新ドラをめくる。
- ドラ表示牌が西だった場合、抜かれた北は抜きドラかつ通常ドラとしてダブルカウントする。

注）４枚目の北でロンされた場合、新ドラはめくらない。槍槓でも同様。

注）前項で述べたが、手牌内の北（北とみなしたマイティー牌含む）と抜かれた北の枚数の合計が５枚以上になるケースも認められない。（前項の「・手牌内で、同じ牌が5牌以上存在するような取り方はできない。」より）

### 暗槓 明槓
槓をした場合、ドラ表示は先めくりとなる。
### 付加カン
他家のポンにカンできる。ロンされたら槍槓が付く。槍槓が発生した際、支払う点数の配分はカンされた側とカンした側と折半になる。チップは両方からもらえる。例えば、カンされたプレイヤーが付加カンへの槍槓で和了した場合、もらえる点数は半分になる。
### 箱リー
持ち点が0点で立直を行う場合、その時点までに自分が切った牌を全て裏返した上で、その上から箱を被せ、次巡以降の自分のツモ切り牌も裏返して切る。この場合、フリテンは全解消される。ハコリー中に他の家が白フラッシュをした場合は白フラッシュが有効な裏向きでツモ切りとなりロンもできず和了することができない。ハコリーを行った者が白を切った場合でも裏返しで捨てるため白フラッシュは発動しない([白フラッシュ](#白フラッシュ)については後項を参照）。ハコリーで上がれず、その局が流局した場合、立直者はトビ（1トビ）になる。トビ賞はその時点でトップのものになる。これは途中流局で流局した場合も同様。
### ダブドラ
・赤ドラがドラ表示牌で、ドラはダブドラとする。（つまり6ピンか6ソー）　

・西がドラ表示だった場合、[北](#北)を参照。

### チョンボ
・チョンボ→チップ2枚オールとしその局を初めからやり直す。（誤ロン、誤ツモの発声のみの場合は上がり放棄、倒牌した場合はチョンボ）

・局進行中の山の白の見せ牌もチョンボとする。

・多牌は発覚でチョンボ、少牌は上がり放棄。

## 役について
基本は通常の麻雀に準ずる。ただし、点数計算が違うものやオリジナル役が存在する為以下に記載する。
### 小三元
小三元は役満。
### 大三元
大三元は3倍役満。
### 流し満貫(流し役満)
流し満貫は役満とする。
### 鬼将軍
マイティー牌を使用せずに役が立直のみのロンで他家から上がった場合、その和了は三倍満となる。ツモ和了、 一発などの偶然役や裏ドラ, 北抜きなどがある場合は鬼将軍とならない。また、通常のチップ清算に加え祝儀でチップ5枚オールとなる。([祝儀](#祝儀)については後項を参照)

役満については以下を採用している。なお、役満同士は複合する。 

役満）　　国士無双、四暗刻、天和、地和、緑一色、字一色、小四喜、清老頭、九蓮宝燈、四槓子、小三元、流し役満  
2倍役満）大四喜、四暗刻単騎、国士無双13面待ち、純正九蓮宝燈  
3倍役満）大三元  


## 局の終わり
### 和了
ダブロンありである。
以下の手順で行う。

1, 手配の確定  
- 白の取り方が複数ある場合、裏、アリスをめくる前に確定させる。忘れた場合は、手牌のみで見た時の高めがあればそのように取り、変わらなければ、発覚時で安目どり。
- 手牌内（抜きドラを含む）で、同じ牌が5牌以上存在するような取り方はできない。

2, 点数計算  
必要な場合裏ドラを捲り、点数計算を行う。ツモ損あり(点数計算は4人麻雀に準ずる）

3, チップ計算  
ツモ上がり時は他家全員が、ロン上がり時は振り込んだ他家がチップを支払う。
以下の場合にチップが移動する。

- 一発 - 一発で上がった場合、チップが1枚発生する
- 裏ドラ - 裏ドラが一つ載るごとにチップが1枚発生する
- リアル - マイティー牌を使用せずに和了した場合、チップが2枚発生する（マイティー牌として白を使ってない場合、リアルはつく）
- 枚場 - その時点の枚場の数だけチップが発生する。`(ex. 2枚場の場合はチップが2枚発生する）`枚場ルールに関しては[枚場](#枚場)を参照。
- アリス-リーチ和了時、次の上ヅモ牌をめくる。めくった牌が手にあればさらに次の上ヅモ牌をめくることができる。これをめくった牌が手牌と被らなくなるまで続行。被った分だけチップが加算される。
（注,アリスに赤ドラが出てきても1枚とカウントする。）  
（注,白をマイティー牌として使用している場合はその牌として扱い白としては扱わない。）

チップが不足し払えなくなった場合は、貸し付けて調整する。

4, 祝儀  
リアル役満または鬼将軍和了で発生する。和了で5枚オール。n倍役満については祝儀が5n枚発生する。その後、ツモなら他家全員、ロンなら放銃者にリアルなど通常のチップ清算を行う。n倍役満について祝儀が5n枚発生する。

また、 数え役満(13飜以上)を和了した場合、和了時の翻数で13を超えた分だけチップが発生する。
```
ex)14翻で和了した場合はチップが1枚発生する
```
5, 枚場  
親連荘、または流局で発生する。4人麻雀のn本場と発生条件は同じだが、点棒ではなくチップが加算される。

## 流局
- 九種九牌（ただしマイティ―牌を他の牌とみなした九種九牌は不可）
- 四槓流れあり。また５枚目の新ドラがめくれる状況も流局とする。
- 王牌として残すのは最初のドラ表示牌含めて5列残るようにする。（最大で18枚、最低で10枚)
- チョンボ→チップ2枚オールとしその局を初めからやり直す。（誤ロン、誤ツモの発声のみの場合は上がり放棄、倒牌した場合はチョンボ）
- 局進行中の山の白の見せ牌もチョンボとする。

## 半荘の終わり
南三局が終わるか、誰かがトビになった場合にその半荘は終わる。
### 清算について
25000点持ち30000点返し　
順位点　1位 +15　2位 -5　3位 -10

#### トビ
点棒が0点未満になった時に発生する（0点の場合は続行）。  
トビ賞は10。  
点棒は上家順に払っていくため、ノー聴罰符やダブロンでトビが起こった場合、0点未満にしたプレイヤーにトビ賞が加算される。
箱下精算は行われる。
#### 焼き鳥
半荘を通して一度も和了がないまま、半荘が終了した場合（跳び終了含む）の原点として発生する。  
20点を焼き鳥の人数で割り各々の点から引かれる。その分は非焼き鳥の点に均等になるように加算。   
全員が焼き鳥の場合は点数の移動は起こらない。
```
ex) A,Bが焼き鳥、Cのみ和了しAがトビ終了。→
A:nトビ、(Aの素点)-10(準位点)-10(トビ)-10(焼き鳥)=-60-n、
B:(Bの素点）-5(準位点)-10(焼き鳥)、
C:(Cの素点)＋30(準位点、オカ)＋20(焼き鳥ボーナス)＋10(トビ賞)
```
## セットの終了
### トップ賞、ラス賞
6半荘の1セット以上の時、6連続トップ、または6連続ラスはセット終了時チップがもらえる。それぞれ5枚オール、2枚オール。
